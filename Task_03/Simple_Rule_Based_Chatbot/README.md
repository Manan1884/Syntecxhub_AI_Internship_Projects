# Simple Rule-Based Chatbot

## Project Description

This project implements a Simple Rule-Based Chatbot using pattern matching and a predefined knowledge base. The chatbot interacts with users through the console, recognizes user intents such as greetings, help requests, and small talk, and responds using predefined rules. It also answers domain-specific questions from a small knowledge base and maintains a conversation history during the session.

---

## Features

- Interactive console-based chatbot
- Pattern matching using predefined rules
- Supports multiple intents
  - Greeting
  - Help
  - Small Talk
  - Exit
- Domain-specific knowledge base
- Conversation history logging
- User-friendly interface

---

## Project Requirements Covered

- Build a conversational chatbot using pattern matching

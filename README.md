## 🚀 Projects

### 📌 Task 01 – Maze Solver using A* Search

**Objective:** Implement the A* Search Algorithm to find the shortest path in a maze.

Features

- Represented a maze as a grid of nodes
- Implemented the A* Search Algorithm
- Used Manhattan Distance heuristic
- Found the shortest path
- Handled unreachable cases
- Console-based maze visualization

---

### 📌 Task 02 – Rule-Based Expert System

**Objective:** Develop an expert system capable of drawing conclusions using rule-based reasoning.

Features

- Built an IF-THEN rule engine
- Implemented Forward Chaining
- Supported multi-step inference
- Logged inference steps
- Interactive console application

---

### 📌 Task 03 – Simple Rule-Based Chatbot

**Objective:** Build a conversational chatbot using pattern matching and predefined rules.

Features

- Pattern matching for user queries
- Greeting, Help, and Small Talk intents
- Rule-based response generation
- AI knowledge base for domain-specific questions
- Interactive Command-Line Interface (CLI)
- Conversation history logging

---

### 📌 Task 04 – Sentiment Analysis Tool

**Objective:** Build a Machine Learning model to classify text as Positive or Negative.

Features

- Loaded labeled text data from a CSV dataset
- Text preprocessing and tokenization
- TF-IDF feature extraction
- Sentiment classification using Multinomial Naive Bayes
- Model evaluation using Accuracy and F1 Score
- Interactive Command-Line Interface (CLI)
